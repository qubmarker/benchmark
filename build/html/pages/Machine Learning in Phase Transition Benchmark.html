Dataset¶

Ising model¶

Traditional 2-dimensional Ising model with periodic boundary condition.

\[H=-J\sum_{\langle i,j\rangle}s_i s_j, \quad s_i,s_j\in \{-1,1\},\langle i,j\rangle \text{ are nearest neighbors}\]

Here we assume \(J=1\), so there is a phase transition from order phase to disorder phase in critical temperature point \(T_c=\frac{2}{\ln (1+\sqrt 2)}\approx2.269\).

There are five Ising model datasets, with size 8x8, 12x12, 16x16, 20x20 and 24x24. The dataset is generated by cluster algorithm (Wolff algorithm).

Data points are sampled from 4000 temperature points uniformly divided from \(0.02\) to \(4.02\). For each temperature there are 10 sampled data points. Each point has 5 columns: Temperature(‘temperature’), Configuration(‘conf’), Specific Heat Capacity(‘cv’), Susceptibility(‘sus’) and Average magnetization(‘m’).

Q-Potts model¶

Size 16 Q-Potts model with periodic boundary condition.

\[H_p =-J_p \sum_{\langle i,j\rangle} \delta(s_i,s_j), \quad s_i,s_j\in \{1,2,\cdots,q\}\]

Here we assume \(J_p=1\), so there is a phase transition from order phase to disorder phase in critical temperature point \(T_c=\frac{1}{\ln (1+\sqrt q)}\).

There are four q-potts model datasets with size 16 and q from 3 to 6. The dataset is generated by cluster algorithm (Wolff algorithm).

Data points are sampled from 4000 temperature points uniformly divided from \(0.02\) to \(3.02\). For each temperature there are 10 sampled data points. Each point has 5 columns: Temperature(‘temperature’), Configuration(‘conf’), Specific Heat Capacity(‘cv’), Susceptibility(‘sus’) and Average magnetization(‘m’).
