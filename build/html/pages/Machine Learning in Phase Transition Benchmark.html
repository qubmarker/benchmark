Ising model¶

Traditional 2-dimensional Ising model with periodic boundary condition.

\[H=-J\sum_{i,j}s_i s_j\]

, \(s_i,s_j \in \{-1,1\}\), \(<i,j>\) are nearest neighbors.

Here we assume \(J=1\), so there is a phase transition from order phase to disorder phase in critical temperature point \(T_c=\frac{2}{\ln (1+\sqrt 2)}\approx2.269\).

We have three Ising model datasets, with size 8x8, 12x12 and 16x16. The dataset is generated by cluster algorithm (Wolff algorithm).

Each dataset concludes 5 columns: Temperature(‘temperature’), Configuration(‘conf’), Specific Heat Capacity(‘cv’), Susceptibility(‘sus’) and Average magnetization(‘m’)

Average magnetization: \(m=\sum_i s_i /N\)

Susceptibility:

\[\chi=\frac{(\langle m\rangle^2-\langle m^2 \rangle)}{T}\]

Specific Heat Capacity: \(C_V=\frac{(\langle m\rangle^2-\langle m^2 \rangle)}{T^2 N}\)

Each dataset has 216320 data points. Temperature range is from 0.001 to 4.499.

For temperature in \(T\in [2.2,2.3]\), the temperature gap is 0.0001. For temperature out of the range, the gap is 0.001.

Each temperature has 40 configurations. In each temperature, the configuration is initialized by random, and after 50000 Monte Carlo steps to evolve. Then each 5000 Monte Carlo steps will give 1 configuration.

XY model¶

Traditional 2-dimensional XY model with periodic boundary condition.

\(H=-J\sum_{i,j} \vec s_i \cdot \vec s_j\), \(\vec s_i=(\cos \theta_i , \sin \theta_i)\) ,\(<i,j>\) are nearest neighbors.

There is not a conventional phase transition present that would be associated with symmetry breaking. The system does show signs of a transition from a disordered high-temperature state to a quasi-ordered state below some critical temperature, called the Kosterlitz-Thouless transition.

Approximately, there has a critical temperature \(\frac{2T_c}{J}\ln \frac{2T_c}{J}=1\) and \(T_c/J\approx 0.8816\).

The dataset concludes 4 columns: Temperature(‘temperature’), Configuration(‘conf’), Average magnetization(‘m’) and squared magnetization (ms)

average magnetization: \(m=|\langle(\sum_i \cos\theta_i , \sum_i \sin\theta_i)\rangle|/N\)

squared magnetization: \(m^2 =\frac{1}{N^2}\langle(\sum_i \cos\theta_i)^2+(\sum_i\sin\theta_i)^2 \rangle\)

The dataset has 200000 data points. Temperature range from 0.02 to 5.02, temperature gap is 0.001, each temperature has 40 configurations. In each temperature, the configuration is initialized by random, and after 50000 Monte Carlo steps to evolve. Then each 5000 Monte Carlo steps will give 1 configuration.

Ising model with magnetic field¶

Traditional 2-dimensional Ising model in magnetic field with periodic boundary condition.

\[H=-J\sum_{i,j}s_i s+h\sum_i s_i\]

, \(s_i,s_j \in \{-1,1\}\), \(<i,j>\) are nearest neighbors.

At a constant temperature, the model will have or not have a phase transition when \(h\) varies from minus to plus. At \(T<T_c\), there is a phase transition at \(h=0\) and a delay phenomenon. At \(T>T_c\) there is no phase transition.

The dataset has three columns. Configuration (‘conf’), out magnetic field (‘mag’) and magnetization (‘m’).

Average magnetization: \(m=\sum_i s_i /N\)

The dataset has 200000 data points. out magnetic field range from -6.02 to 6.02, field gap is 0.0024, each temperature has 40 configurations. In each magnetic field, the configuration is initialized by random, and after 50000 Monte Carlo steps to evolve. Then each 5000 Monte Carlo steps will give 1 configuration.

Ising model with unequal coupling¶

Traditional 2-dimensional Ising model with unequal coupling with periodic boundary condition.

\[H=-J_x\sum_{i,j}s_{i,j} s_{i+1,j}-J_y\sum_{i,j}s_{i,j} s_{i,j+1}\]

, \(s_i,s_j \in \{-1,1\}\)

Here we assume \(J_x=1\), so there is a phase transition from order phase to disorder phase in critical temperature point \(T_c \)sinh(2/T_c)sinh(2J_y/T_c)=1`.

The dataset has three columns. Configuration (‘conf’), Temperature(‘temperature’) and magnetization (‘m’).

Average magnetization: \(m=\sum_i s_i /N\)

The dataset has 200000 data points. Temperature range from 0.02 to 6.02, field gap is 0.0012, each temperature has 40 configurations. In each magnetic field, the configuration is initialized by random, and after 50000 Monte Carlo steps to evolve. Then each 5000 Monte Carlo steps will give 1 configuration.
